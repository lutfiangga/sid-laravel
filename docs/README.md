# Dokumentasi SID

Selamat datang di dokumentasi teknis SID (Sistem Informasi Desa). Dokumen ini memberikan panduan mendalam tentang struktur kode, arsitektur, dan fungsionalitas setiap modul.

## Daftar Isi

1. **[Arsitektur Dasar](./architecture.md)**
    - Memahami pemisahan antara `app/Core` dan fitur di `Modules/`.
    - Repositori, Service, dan Pattern yang digunakan.

2. **[Modul Kependudukan](./modules/population.md)**
    - Pengelolaan Data Penduduk & Kartu Keluarga.
    - Struktur Wilayah (Dusun, RW, RT).

3. **[Modul Keuangan](./modules/finance.md)**
    - Manajemen Periode Anggaran.
    - Mata Anggaran (COA).
    - Pagu Anggaran (RAB) & Transaksi Realisasi.

4. **[Modul Korespondensi (Surat)](./modules/correspondence.md)**
    - Pengaturan Jenis Surat.
    - Alur Permohonan & Persetujuan Surat.

5. **[Modul Layanan Publik](./modules/public-service.md)**
    - Pengumuman Desa.
    - Daftar Aparatur (Pamong).
    - Sistem Pengaduan & Aspirasi Warga.

6. **[Modul Sistem](./modules/system.md)**
    - Role-Based Access Control (RBAC).
    - Pengaturan Global & User Management.

---
*Dokumentasi ini dikelola secara internal untuk mempermudah pengembangan dan pemeliharaan platform SID.*
